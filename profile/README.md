# yodo.gg
### Silly name; serious service.

![https://github.com/yodo-gg/.github/profile/docs/images/logo.png]

## Goals

- As anonymous as we can get away with.
- As cheap as we can get it.
  - For us and the users. 
- Reasonably priced add-ons.
  - Web UI.
- Open Source.
  - No gatekeeping ideas.
  - If you want to host it yourself, go for it!
  - We welcome contributions!
- Privacy-focused payment methods preferred.
  - If you want to buy us a coffee we won't mind!
- Have fun! (Do no harm but take no shit.)
